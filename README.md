# 🌱 Gamified Environmental Education Platform

## 📌 Overview
This project is a web-based platform designed to promote environmental awareness through gamification. Users can participate in quizzes, activities, and challenges while tracking their progress and performance.

---

## 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 📊 Dashboard for Students and Admin
- 🧠 Quiz System with Levels (Easy, Medium, Advanced)
- 🏆 Leaderboard and Performance Tracking
- 🎯 Environmental Activities & Challenges
- 📈 Reports and Progress Monitoring
- 🎥 Video-based learning activities

---

## 🛠️ Technologies Used

- Backend: Python (Flask)
- Frontend: HTML, CSS (Jinja Templates)
- Database: SQLite

---

## 📂 Project Structure

app.py → Backend logic  
database.db → Database  

templates/ → HTML pages  
static/ → CSS, images, videos  

---

## ⚙️ How to Run

1. Run the app:
python app.py

2. Open browser:
http://127.0.0.1:5000
